Face Recognition

I built an application that allows users to pass an image to an application platform interface (API) and able to send back results of coordinates of people's faces. The image is sent back and displayed for the user with their submitted picture. I also integrated a sign in and register function for user security of the application. The technologies that were used to build this project was React, Node, Postgres and used Render.com to host application.
